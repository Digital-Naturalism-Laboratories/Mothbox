# Assembly
This newest version of the Mothbox is super fast to assemble! You only need

* Mothbox PCB
* Arducam 64mp camera (owlsight)
* Raspberry Pi 5
* 32 MB A1 SD card (and better, more GB, A2 speed)
* Battery

and you are set!
